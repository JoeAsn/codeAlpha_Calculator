# 🧮 Calculator App

A clean and responsive **Calculator Web Application** built as part of my Front-End Development Internship at **[CodeAlpha]**.

---

## 🚀 Features

- Basic arithmetic operations: Addition, Subtraction, Multiplication, Division
- Clear / Reset functionality
- Responsive design — works on all screen sizes
- Clean and intuitive user interface
- Keyboard-friendly interactions

---

## 🛠️ Built With

- **HTML5** — Structure
- **CSS3** — Styling & Responsiveness
- **JavaScript (Vanilla)** — Logic & DOM Manipulation

---

## 📁 Project Structure

```
calculator-app/
├── index.html       # Main HTML file
├── style.css        # Stylesheet
└── script.js        # JavaScript logic
```

---

## ⚙️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/calculator-app.git
   ```

2. Navigate to the project folder:
   ```bash
   cd calculator-app
   ```

3. Open `index.html` in your browser — no installation needed!
